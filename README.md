# Computing Historian Agent 🤖

AI agent built with Microsoft Foundry that answers questions 
about the history of computing and AI.

## Technologies
- Microsoft Azure Foundry
- GPT-4.1-mini
- Python
- Azure AI Projects SDK

## Setup
1. Clone the repo
2. Create a `.env` file with your `AGENT_ENDPOINT`
3. Run `pip install azure-ai-projects>=2.0.0 openai`
4. Run `az login`
5. Run `python agent.py`
